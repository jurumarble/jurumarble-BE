## 프로젝트 소개 💡
- 주루마블(Jurumarble)은 우리술 투표부터 우리술과 함께 할 여행지 맛집 추천까지,여행 및 우리술 경험을 공유하고 아카이빙하여 즐거운 여행을 돕는 우리술 여행 서비스 입니다
